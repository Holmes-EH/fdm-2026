### **1. Description du métier**

**Titre** : _"Développeur web / Ingénieur logiciel : le magicien du numérique !"_

**Contenu** :

- _"Un développeur, c’est quelqu’un qui crée des sites web, des applications ou des jeux vidéo en écrivant du code, un peu comme une recette pour l’ordinateur."_
- **Exemples concrets** :
    - _"Quand tu utilises TikTok, YouTube ou un jeu mobile, c’est grâce au travail de développeurs !"_
    - _"Moi, par exemple, je travaille sur un logiciel de production vidéo. Captation, Ralentis, Enregistrements, ..."_
- **Outils** : 
	- Editeur de code
	- Google
	- Documentation
	- Intelligence Artificielle
- **Pourquoi c’est cool** :
    - _"On peut travailler sur des projets variés : réseaux sociaux, robots, outils pour les médecins, etc."_
    - _"C’est un métier où on apprend tous les jours."_
